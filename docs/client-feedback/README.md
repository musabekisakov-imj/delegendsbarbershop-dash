# Client Feedback

Записи разговоров, скриншотов с правками, решений принятых клиентом.

**Когда писать:** после каждого звонка или ревью, где появились новые требования или правки.

**Как именовать:** `YYYY-MM-DD-тема.md`
Пример: `2026-05-09-header-redesign-feedback.md`

**Главное:** фиксируй action items сразу — через день детали смажутся.
